# KreativRoom

KreativRoom is a Next.js 16 landing page for a premium video editing and personal branding agency. The site focuses on service positioning, portfolio-style showcases, testimonials, pricing, and multiple call-to-action sections.

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Framer Motion
- TypeScript

## Local Development

```bash
npm install
npm run dev
```

The app runs locally at `http://localhost:3000` by default.

## Production Checks

```bash
npm run lint
npm run build
```

## Notes

- Environment files are ignored through `.gitignore`.
- Runtime and local asset scratch files in the project root are not intended for version control.
